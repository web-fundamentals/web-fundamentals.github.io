How To Create a Repo

  1. Go to GitHub.
  2. Click on New repository.
  3. Type in your repo name (all lowercase, no spaces) and hit Create repository.
  4. In the next screen, copy the git link of your repo.
  5. Open iTerm.
  6. Navigate (cd) to your web-fundamentals folder
  7. Type git clone and then paste in the url you copied. This will create a folder with that name. That's your repo.
  8. cd into that repo and you're ready to roll!
