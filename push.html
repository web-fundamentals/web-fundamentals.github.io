How To Push To Git and Publish To Your gh-pages Branch

  1. cd into the repo you're trying to push up.
  2. Once you're in the repo, type
    git add .
    git commit -m "add a commit message here about what you did"
    git push origin master
    That pushes your repo up to your master branch. It's not live on the internet at this point. Your files just live on your github profile page.


tl;dr :: quick and dirty

cd into your repo

git add . to add your files

git commit -m "message here" to commit your files

git push origin master to push your files



Do you want to publish this live?

  1. Make sure you're in the repo you want to publish.
  2. If this is your first time publishing this live, you'll need to create a gh-pages branch.
  3. Type git branch gh-pages
    This creates your gh-pages branch. If you type git-branch at this point, it should show you
    gh-pages
    *master
  4. Now we need to switch to our gh-pages branch.
    Type git checkout gh-pages
  5. Now you have to merge in your master branch. Type git merge master
  6. Then push up your code. Type git push origin gh-pages
  7. Always switch back to your master branch at this point. git checkout master
  8. Now you don't have to create this branch again. You just have to switch to it, merge in your master branch, publish, and switch back to master.


tl;dr :: quick and dirty

git branch gh-pages to create the branch

git checkout gh-pages to switch to it

git merge master to merge the master branch

git push origin gh-pages to publish it

git checkout master to switch back to master
